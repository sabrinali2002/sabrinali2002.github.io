About Me

I'm a Full Stack Software Engineer at IBM in San Jose, California, working on IBM DataStage — a product that helps non-technical users handle data from databases and file systems more easily through a visual drag-and-drop interface that generates SQL queries and other data pipeline functionality under the hood.

I'm on the API team, so most of my work happens behind the scenes. I help optimize the RESTful APIs that power DataStage's platform and build out new connector functionalities — crafting endpoints that handle user inputs, process them efficiently, and return relevant data or responses from backend systems and third-party integrations. I graduated from Cornell University in 2024 with a Bachelor of Science in Computer Science.

Before IBM, I interned at Publicis Sapient (Summer 2023), where I designed and developed an AI chatbot for a car dealership company, compatible with both web and mobile, using React, Python, and Google Cloud Vertex AI. I integrated APIs to aggregate dealership data — inventory, distances, and locations — enabling personalized user queries and improving customer engagement.

Outside of work I love playing piano, violin, and guitar, baking, drawing, and building games. I'm always looking for fun side projects to sink time into!
